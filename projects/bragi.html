Project Overview

BRAGI is a groundbreaking AI-powered video generator that empowers creators, marketers, and businesses to produce professional-grade videos with minimal effort. By integrating advanced AI models, BRAGI automates script generation, scene composition, and editing, allowing users to focus on their creative vision. This project is a testament to my growth as a developer, combining full-stack development, AI integration, and performance optimization to deliver a seamless user experience.

Building on my experience from the Subtruck AI project, I enhanced my skills in full-stack development, API creation, and AI model integration. Additionally, I’m excited to share that an iOS app for BRAGI is currently in development, leveraging Swift to bring this powerful tool to mobile users. BRAGI is not just a project—it's a showcase of innovation, optimization, and my passion for creating impactful solutions.

Skills and Technologies

Building on Previous Experience

BRAGI builds on the foundation I established with Subtruck AI, where I first explored app development and AI integration. For BRAGI, I utilized Laravel Livewire and Tailwind CSS to create a dynamic, responsive frontend and a robust backend, significantly improving the user experience and performance. Livewire enabled real-time updates without page reloads, while Tailwind CSS ensured a modern, polished design that scales beautifully across devices.

A major milestone in this project was learning to create and integrate APIs. I developed custom APIs to connect BRAGI with open-source AI models like Wan 2.1, Kling, Google Gemini, Mochi, and others, enhancing the quality and creativity of the generated videos. Using Python, I handled data preprocessing, API requests, and fine-tuning of AI outputs, ensuring seamless integration and high-quality results.

Additionally, I focused on performance optimization by learning advanced hosting techniques. I optimized the server infrastructure to ensure BRAGI runs smoothly and efficiently, delivering fast load times and a responsive experience for users. This project also marks my first step into iOS app development, as I’m currently working on a mobile app for BRAGI using Swift, bringing the platform’s capabilities to iOS users.
